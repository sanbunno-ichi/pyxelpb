# Pyxel Bobbles

## 概要
- ゲーム内容はパズルボブルタイプのゲーム。  
- 設定にて、マルチカラーと16色モードが選べます。  
- 効果音は、frenchbreadさん作成の「Pyxel RPG SE パック」を使用しています。
